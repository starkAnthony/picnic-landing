# Sunlit Picnics — Landing Page

A React landing page for a curated picnic business, built with Vite.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

## Customize

- **Brand name & copy** — Edit components in `src/components/`
- **Packages & pricing** — `src/components/Packages.tsx`
- **Colors** — CSS variables in `src/index.css`
- **Contact info** — `src/components/Footer.tsx` and `Booking.tsx`

The booking form shows a success message locally. Connect it to your backend or a service like Formspree when you are ready to go live.
